HOW TO ADD SRINGERI TEMPLE IMAGES
===================================

Place your Sringeri temple photos here with EXACTLY these filenames:

  temple1.jpg  —  Sharadamba Temple (main gopura / front view)
  temple2.jpg  —  Sharada Peetham (full complex aerial/wide view)
  temple3.jpg  —  Vidyashankara Temple
  temple4.jpg  —  Tunga River / Ghat view
  temple5.jpg  —  Temple interior / mandapa / pillars

WHERE TO DOWNLOAD REAL SRINGERI TEMPLE IMAGES:
------------------------------------------------
1. Google Images  →  search "Sringeri Sharadamba Temple"
                      or "Sringeri Sharada Peetham"
                      or "Vidyashankara Temple Sringeri"

2. Official site  →  https://sringeri.net

3. Wikipedia      →  https://en.wikipedia.org/wiki/Sringeri_Sharada_Peetham
                      (right-click images → Save As)

RECOMMENDED SIZE:  1920 x 1080 px or larger
FORMAT:            .jpg preferred (smaller file size)

After placing the images, refresh your browser — they will appear immediately.
Until images are placed, a dark saffron gradient is shown as placeholder.

FOR FRONTEND (React):   place in  Frontend/public/images/temple/
FOR BACKEND  (static):  place in  Backend/parking/src/main/resources/static/images/temple/
